# END-TO-END-DATA-SCIENCE-PROJECT

## Description

This project demonstrates a complete Data Science workflow from model training to deployment using Flask.

## Features

- Data Preprocessing
- Model Training
- Model Serialization
- REST API Deployment
- Prediction Endpoint

## Technologies Used

- Python
- Flask
- Scikit-Learn

## Installation

pip install -r requirements.txt

## Run Model Training

python model_training.py

## Run API

python app.py

## API Endpoint

POST /predict

Example Input:

{
    "feature1": 5.1,
    "feature2": 3.5,
    "feature3": 1.4,
    "feature4": 0.2
}

## Deliverable

An end-to-end data science application demonstrating data processing, model training, and deployment using Flask.
